# Love Letter Web App

A cute, mobile-friendly web app for creating a sweet digital love letter with:

- a pastel envelope that opens with a soft animation
- editable greeting, message, and signature fields
- theme colors and sticker decorations
- optional photo upload
- preview, shareable-link, and download options

## Open it locally

Open the file in a browser, or serve the folder with a simple local server.

Example:

- python -m http.server 8000
- then visit http://127.0.0.1:8000/index.html
